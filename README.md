# Creative Freedom Is Not the Same as Communication Judgment

GitHub Pages-ready microsite for the Be Positive 24 recruitment ad case-study article.

## Files

- `index.html` - main web page
- `assets/original-be-positive24-recruitment-ad.png` - original case-study ad visual
- `assets/comment-map-art-context-fit-poster.png` - summary poster created for the analysis
- `assets/creative-freedom-communication-judgment-case-study.pdf` - PDF version of the article

## Publish on GitHub Pages

1. Create a new public GitHub repository.
2. Upload the contents of this folder, not the ZIP itself.
3. Go to **Settings > Pages**.
4. Select **Deploy from a branch**.
5. Choose **main** and **/root**.
6. Save and wait for the live Pages link.
